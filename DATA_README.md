# Aadhaar Harm Database
Documentation of harms caused by Aadhaar-based authentication failures and exclusions.

## Data Files
- `aadhaar_harm_cases.json` — Full dataset (66+ cases)
- `aadhaar_harm_cases.csv` — Tabular export
- `schema.sql` — SQLite schema

## Fields
- `id` (e.g., A1, A2)
- `section` (A–I categories)
- `case_type`
- `name`
- `details` (entitlement denied / mechanism)
- `outcome` (result)
- `gender`, `age`, `location`, `year`
- `notes`
- `source` (citation)
